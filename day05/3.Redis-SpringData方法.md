### Redis基础指令对应的方法

*java端插入的值的类型和Redis端并无关系，java可插入任意类型的值，Redis端都会转换成String*
*以下数据类型操作，均可以创建实例，这里只是省略这一步骤*,比如：
```java
    ValueOperations valueOperations = redisTemplate.opsForValue();
```
----------------------------------------------------------------------
**String**

```java
                                // "city", "北京"
    redisTemplate.opsForValue().set(key, value);                //SET
    redisTemplate.opsForValue().get(key);                       //GET
    redisTemplate.opsForValue().set(key, value, 持续时间, 单位); //SETEX
    redisTemplate.opsForValue().setIfAbsent(key, value);        //SETNX
```
----------------------------------------------------------------------
**Hash**

```java                            
                            // "person", "name", "Tom"
    redisTemplate.opsForHash().put(key, field, value);         //HSET
    redisTemplate.opsForHash().get(key, field);                //HGET
    redisTemplate.opsForHash().delete(key, field);             //HDEL
    redisTemplate.opsForHash().keys(key);                      //HKEYS
    redisTemplate.opsForHash().values(key);                    //HVALS
```
----------------------------------------------------------------------
**LinkedList**

```java
    redisTemplate.opsForList().leftPush("list", "1");               //LPUSH
    redisTemplate.opsForList().leftPushAll("list", "2", "3", "4");  //LPUSH
    redisTemplate.opsForList().range("list", 0, -1);                //LRANGE
    redisTemplate.opsForList().rightPop("list");                    //RPOP
    redisTemplate.opsForList().size("list");                        //LLEN
```
----------------------------------------------------------------------
**Set**

```java
    redisTemplate.opsForSet().add("set", "1","2","3");          //SADD
    redisTemplate.opsForSet().members("set");                   //SMEMBERS
    redisTemplate.opsForSet().size("set");                      //SCARD
    redisTemplate.opsForSet().intersect("set1", "set2");        //SINTER
    redisTemplate.opsForSet().difference("set1", "set2");       //SDIFF
    redisTemplate.opsForSet().union("set1", "set2");            //SUNION
    redisTemplate.opsForSet().remove("set", "1", "2");          //SREM
```
----------------------------------------------------------------------
**zSet**

```java
redisTemplate.opsForZSet().add("zset", "a", 分数);                   //ZADD
redisTemplate.opsForZSet().range("zset", 0, -1);                    //ZRANGE
redisTemplate.opsForZSet().incrementScore("zset", "a", 分数增长量); //ZINCRBY
redisTemplate.opsForZSet().remove("zset", "a");                    //ZREM
```
----------------------------------------------------------------------
**通用命令**

```java
    redisTemplate.keys("*");        //KEYS pattern  获取指定模式的key(Set集合，可遍历)
    redisTemplate.keys("abc*");
    redisTemplate.hasKey(key);      //EXISTS key
    redisTemplate.type(key);        //TYPE key
    redisTemplate.delete(key);      //DEL key
```
